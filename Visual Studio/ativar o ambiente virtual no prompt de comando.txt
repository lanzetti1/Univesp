No prompt de comando:

crie um diretório : mkdir <nome do diretório>.
	para remover:rmdir <nome do diretorio> /s /q  (/s/q - apaga o conteúdo interno do diretótio)

entre no diretório: cd <nome do diertório>

crie o ambiente virtual: virtualenv <nome do ambiente>

ative o ambiente virtual: <nome do ambiente>\scripts\activate.bat
	instale o programa que vai utilizar: pip3 install <nome da aplicação>
	para desativar:deactivate

crie uma página de teste: nano <nome da página>.py
	para testar a página: set <nome do app>_APP =<nome da página>
	para rodar a página em modo de desenvolvimento: set <nome do app>_ENV = devlopment
	rode a aplicação e verifique a página criada: <nome da aplicação> run

abra um navegador e cole a endereço gerado.
